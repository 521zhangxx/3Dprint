# Prusa i3资料
**教程以自己思考为主，技术支持为辅！**
本技术支持为额外的服务，无期限完全免费，态度差者将拒绝提供技术支持！
目录介绍：
1.安装指导
包含安装文档！！！
2.驱动程序
包含打印机连接电脑时所需安装的驱动程序！如果安装失败请安装驱动精灵尝试解决！
3.软件
包含电脑端需要安装的控制软件RepetierHost_1_5_6，以及建模，切片等软件！
4.模型
包含提供的切片好的gcode文件，以及网络收集的大量STL文件！
5.固件源码
包含固件源码及编译好的hex文件，供烧录（刷机），事实上，我们出厂前已烧录。
6.主板原理图
包含主控板原理图，供专业人士参考！
7.建模教程
包含犀牛和soildworks两个建模软件的视频教程。
8.网络教程收集
包含楼主觉得不错的网络教程或者技术贴！
